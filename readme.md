# Fallout Craft

Fallout craft is a small mod which adds a few items from the fallout series to minecraft. It's not only a Fallout mod, it's a mod for all things Apocalypse.

### Features

- Fallout themed items (Nuka Cola)
- Radiations Water
- Fallout Paintings
